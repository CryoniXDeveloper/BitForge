PREFIXES

Operand 1 (r, mi, mr):
- ws{BITS}b.{OPERAND_1}

Operand 2 (r, i, mi, mr):
- ws{BITS}b.{OPERAND_2}

mval8–64:
- Operand 2 can be any size; if >64 bits use mvalplus
- Cropping: moving bigger value to smaller space removes high bytes (from start)
- Zero-extension: moving smaller value to bigger space pads with zeros
- No custom sizes; use mvalplus for that

mvalplus:
- Operand 2 can be any size
- Cropping: bigger → smaller removes high bytes
- Zero-extension: smaller → bigger pads with zeros
- Slower but flexible

RULE:
- For mval8–mval64, no operand can exceed 64 bits

ERRORS:
- Can disable errors like: @err throw widthSpecifierTooSmallForImmediate false
